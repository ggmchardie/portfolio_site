Description

I worked on Karuna as part of a small team while finishing my degree. Karuna is a Java-based facial recognition smartphone app created with Android Studio. The app's objective is to bring people together after a natural disaster, even if they are separated by a long distance. Karuna searches for people using images and personal information and facial recognition technology. The location of the individual who was looked for in the database can be found.

About Karuna

Karuna typically translates as "compassion" in English and is a concept used in the spiritual paths of Hinduism & Buddhism. A Sanskrit word, meaning "to do" or "to make," indicating an action-based form of compassion. Karuna is the doing of something to alleviate suffering.

Home Page

Upon opening the app the user is presented with the log in page

Login Page

Users can log in or sign up using amazon services

Home Page 2

Once logged in, the user will be redirected back to this home screen.

Menu

The menu slides out from the left and offers the user a range of options

Upload/Search for Someone

An individual can be uploaded or searched for using personal details and photographs.

Match Found

When a match is found it is displayed to the user. Their photo, name, location and vital status can be seen.

Inbox

If the person searched for isn't in the database it can be saved. Once those details are uploaded into the database at a later date, a message will appear in the inbox to notify the search has been fulfilled.

Admin

Admin Users have the ability to add organisations and disasters. Their account can be accessed via the menu.

Add Disaster

Disasters can be added and ended by first responders via admin accounts. This is done for security reasons.
