Description

While completing my degree, I used Balsamiq to create an online booking system for a symposium/conference. Balsamiq is wireframing and mockup software with a high focus on usability Users needed to be able to add and remove event bookings, as well as view information about the convention and each event. This allowed me to hone my UI/UX design talents and learn how to utilise the Balsamiq programme to produce these designs. Clickable designs were used to depict the entire user experience.

Balsamiq

Home Page

The home page directs users to either log in with their details or register for an account.

About Page

The about page provides users with a description of the event underneath a slideshow of relevant pictures.

Help Page

The help page provides users with support. It displays a frequently asked questions section.

Change Language Page

The change language page offers the user a range of options for switching to their native language.

Menu Component

The menu component is designed as a burger menu in order to be easy and intutive on mobile devices.

Register for an Account Page

The register page allows users to register for an account in order to sign up for events. It also allows users access to a user profile where all their personal details and registered events are stored and can be editted.

Registration Confirmation Page

The registration confirmation page lets users know that their account is now active. It then directs them to either their profile or a list of all events.

Cancel Event Alert

This alert prompts users to ensure they are certain that they want to delete an event from their calendar.

List of Events Page

The list of events page displays a list of all events to the user and provides filtering options.

Registration Progress Component

The registration progress component is located in the user's profile and informs them of how many events from each category they have registered for.
