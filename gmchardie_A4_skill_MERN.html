Hot Hot Holes


MERN Stack & Javascript

Purpose

Hot Hot Holes online ordering app was built to give customers a more convenient option and allow the business owners to view all order easily. The app displays the order options, takes in the order and displays it on an admin page in an index.

Home Page

The home page welcomes users to the site and displays some relevant pictures from the company's instagram.

About Page

The about page provides users with a description of the company.

Order Page

The order page displays options to the user and allows them to sumbit their order.

Orders List

The orders list page displays all current orders to the user.

Contact Page

The contact page allows the user to contact the company via a form.
